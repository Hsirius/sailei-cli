### 安装项目

```
npm i -g sailei-cli
```

### 使用

```
sailei create my-project
```

### 支持功能

家中常备

- scss
- eslint
- prettier
- lodash-es
- react-use
- @reduxjs/toolkit
- typescript-plugin-css-modules

可选

- vite4/webpack5
- yapi-to-typescript
- antd
